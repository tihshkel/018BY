# Настройка PDF обложек для экспорта

## Структура папок

PDF обложки для раздела беременности должны находиться в папке:
```
018BY/assets/pdfs/covers/pregnancy/
```

## Маппинг обложек

Маппинг между ID альбома и PDF обложкой находится в файле `018BY/utils/coverPdfMapping.ts`.

### Формат имен файлов

Для беременности используются следующие имена файлов:

- `DB1_твердый переплет.pdf` - для `pregnancy_60` (DB1_0)
- `DB1_пружина.pdf` - для `pregnancy_a5` (DB1_п)
- `DB2_твердый переплет.pdf` - для `pregnancy_db2` (DB2_0)
- `DB2_пружина.pdf` - для `pregnancy_db2_soft` (DB2_п)
- `DB3_твердый переплет.pdf` - для `pregnancy_db3` (DB3_0)
- `DB3_пружина.pdf` - для `pregnancy_db3_soft` (DB3_п)
- `DB4_твердый переплет.pdf` - для `pregnancy_db4` (DB4_0)
- `DB4_пружина.pdf` - для `pregnancy_db4_soft` (DB4_п)
- `DB5_твердый переплет.pdf` - для `pregnancy_db5` (DB5_0)
- `DB5_пружина.pdf` - для `pregnancy_db5_soft` (DB5_п)

## Копирование файлов

1. Откройте папку с исходными PDF обложками: `C:\Users\тихон\OneDrive\Рабочий стол\обложка`
2. Найдите файлы для беременности (DB1, DB2, DB3, DB4, DB5)
3. Скопируйте файлы в папку `018BY/assets/pdfs/covers/pregnancy/` с правильными именами

Если файлы имеют другие имена, обновите маппинг в `018BY/utils/coverPdfMapping.ts`.

## Как это работает

При экспорте альбома беременности:
1. Система определяет ID альбома (например, `pregnancy_60`)
2. Ищет соответствующий PDF файл обложки в маппинге
3. Вставляет PDF обложку в начало экспортируемого PDF
4. Затем добавляет все страницы, отредактированные пользователем

## Примечания

- Если PDF обложка не найдена, система использует изображение обложки как fallback
- PDF обложка вставляется как первая страница без полей
- Размер обложки автоматически подстраивается под выбранный формат экспорта (A4 или A5)
